Featured Project

Exploring NYC Neighborhoods by ZIP Codes

An interactive web application that allows users to analyze and visualize NYC neighborhoods by ZIP codes. The application provides valuable insights into different areas of New York City through comprehensive data visualization.

Technical Highlights:

  • Developed using Python with Streamlit, Folium, and Matplotlib libraries
  • Processed and integrated datasets containing over 10,000 entries using Pandas
  • Implemented mapping functionality using scikit-learn to connect geographical coordinates with ZIP codes
  • Deployed on Streamlit Cloud for seamless user interaction and real-time data visualization
