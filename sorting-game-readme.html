Sorting Objects Game

A C++ project simulating a sorting task, where objects (shapes) are moved between bins to achieve a sorted configuration. The game is interactive and visually displays bins and shapes using Unicode symbols. I developed this game as my final project for CSCI-135, incorporating concepts in object-oriented programming.

GitHub Gist

Features

  • Interactive Gameplay: Players move objects between bins while following specific rules.
  • Custom Algorithms: Validates user moves, updates game state in real-time, and handles edge cases effectively.
  • Efficient Data Storage: Implements 2D vectors to store and manipulate bin data.
  • Visual Interface: Displays shapes and bins using Unicode symbols, with color-coded shapes for better visualization.
  • Debugging & Optimization: Strengthened debugging skills using GDB, ensuring robust performance and optimized memory management.

How to Play

  1. The game starts by displaying the initial state of the bins with shapes.
  2. You are prompted to enter two bin numbers:
    • The bin to move an object from.
    • The bin to move the object to.
  3. Follow the rules:
    • Objects can only be moved to a bin with the same top shape or an empty bin.
    • The destination bin must have enough capacity to hold the moved objects.
  4. The goal is to sort all shapes into separate bins.
  5. To exit the game, enter -1 -1 when prompted.

Technologies Used

C++ Object Oriented Programming Data Structures 2D Vectors GDB Unicode

Project Structure

  • SortGame.hpp: Header file defining the game class and methods.
  • SortGame.cpp: Implementation file containing the game logic.
  • main.cpp: Entry point of the program, initializes the game and calls the play method.

How to Run

  1. Clone the repository: 
    git clone https://github.com/b-mozz/SortGame.git
  2. Compile the project: 
    g++ SortGame.cpp test.cpp -o SortGame --std=c++20
  3. Run the game: 
    ./SortGame
